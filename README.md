# Maharashtra Diaries New

A modern, full-stack travel platform for Maharashtra built with the MERN stack (MongoDB, Express, React, Node.js).

## 🚀 Getting Started

### 1. Prerequisites
- **Node.js**: (v18+)
- **MongoDB**: A local instance running on `localhost:27017` or a MongoDB Atlas URI.
- **MongoDB Compass**: Recommended for visual database management.

### 2. Backend Setup
```bash
cd backend
npm install
npm run dev
```
The backend will run on `http://localhost:5000`.

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
The frontend will run on `http://localhost:5173`.

## ✨ Features
- **User Authentication**: Secure Login & Signup with JWT and bcrypt password hashing.
- **Modern UI**: Dark-themed glassmorphism design with rich animations.
- **Interactive Explore**: Browse, search, and filter destinations by category.
- **Visual Trip Planner**: Create and visualize a timeline itinerary for your travels.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## 🛠 Tech Stack
- **Frontend**: React 19, TypeScript, Vite, React Router, React Hot Toast, React Icons.
- **Backend**: Node.js, Express, Mongoose (MongoDB).
- **Authentication**: JWT (JSON Web Tokens).
- **Styling**: Vanilla CSS with modern Design Tokens (CSS Variables).
